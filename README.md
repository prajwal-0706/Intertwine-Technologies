## Getting Started

First, run the development server:

Step 1: 
```bash
npm install -g json-server

# and

json-server --watch constants/db.json --port 3001

```

Step 2:

```bash
npm install

# and

npm run dev
```



Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
